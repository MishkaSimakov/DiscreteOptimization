# Дискретная оптимизация

## Структура проекта

Для каждой задачи создаётся отдельная ветка с соответствующим именем. Код находится в папке `src`. В папку `tasks`
кладутся файлы заданий. Они большие и их много, поэтому я не загружаю их на GitHub. Например, файлы 1-го задания нужно
положить в папку `tasks/1`, чтобы код мог с ними работать.

Чтобы запустить код, достаточно из корня проекта выполнить:

```shell
mkdir build
cd build
cmake ..
cmake --build . --target {название задачи}
./{название задачи}
```
